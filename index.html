FreePass

Stateless Password Manager

stay safe

Hundred of people get their password stolen everyday. Don't be one of them, IT'S TIME TO GET SECURE. So you've been looking at solutions to this problem and you've come across this nifty solution called "password manager", those are pretty good but they have a big flaw... They store all your passwords in a database that could potentially be vulnerable to an attack. While we don't doubt that a lot of these password managers have top notch security, the only way you can truly be safe is if you use one that never stores your password, that's where FreePass comes in...

how to use

FreePass is currently in development, but when we launch you will be able to use Password as a chromium browser extension, an android mobile app and possibly a windows aplication

FreePass will generate a unique and secure password for you using a ULR, a username, and a master password. You can even use it offline because the generated password is replicable across all your devices
